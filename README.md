## Setup
```bash
$ npm init
$ npm install express
$ node index.js

$ npm install @babel/node
$ npm install @babel/preset-env
$ npm install @babel/core
$ npm start # after babel, npm start is shortcut of command node index.js

$ npm install nodemon -D
```
